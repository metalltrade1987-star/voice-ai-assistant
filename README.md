# 🎙️ Голосовой AI-ассистент с локальной LLM

Мультимодальный голосовой ассистент, работающий **полностью локально** на RTX 3060 (12GB VRAM). 
Умеет слушать, говорить, видеть, запоминать и искать информацию в интернете.

![Python](https://img.shields.io/badge/Python-3.11-blue)
![Ollama](https://img.shields.io/badge/Ollama-Qwen2.5--VL-green)
![License](https://img.shields.io/badge/License-MIT-yellow)

## 🎯 Возможности

- 🗣️ **Голосовое общение** — распознавание речи (Vosk) + синтез (Silero)
- 🧠 **Локальная LLM** — Ollama + Qwen2.5-VL (мультимодальная модель)
- 👤 **Распознавание лиц** — DeepFace + TensorFlow, биометрическая идентификация
- 💾 **Долговременная память** — SQLite (граф сущностей) + ChromaDB (RAG)
- 🌐 **Работа с интернетом** — собственный парсер сайтов, DuckDuckGo, Open-Meteo
- 🎥 **Анализ изображений** — мультимодальная LLM понимает, что видит камера

## 🚀 Установка

```bash
# 1. Клонируем репозиторий
git clone https://github.com/metalltrade1987-star/voice-ai-assistant.git
cd voice-ai-assistant

# 2. Создаём виртуальное окружение
python -m venv .venv
.venv\Scripts\activate

# 3. Устанавливаем зависимости
pip install -r requirements.txt

# 4. Устанавливаем Ollama и модель
# https://ollama.com/download
ollama pull qwen2.5-vl

# 5. Скачиваем модель Vosk (ru)
# https://alphacephei.com/vosk/models

# 6. Запускаем
python main.py

🎓 Ключевые архитектурные решения
1. LLM as Router
LLM сама принимает решения, какое действие выполнить:
Запомнить факт о пользователе
Найти информацию в интернете
Ответить на вопрос
Распознать лицо
2. Гибридная память
SQLite — структурированные данные (граф сущностей и связей)
ChromaDB — семантический поиск по базе знаний (RAG)
3. GPU-оптимизация
Распределение 12GB VRAM между 5+ моделями:
DeepFace: ~2GB
Ollama (Qwen2.5-VL): ~8GB
Silero: ~0.5GB
Остальное: системные нужды

📊 Результаты тестирования
Распознавание речи: ~95% точность (Vosk)
Распознавание лиц: ~98% точность (DeepFace)
Время отклика: 1-3 секунды
Работа в реальном времени: ✅
